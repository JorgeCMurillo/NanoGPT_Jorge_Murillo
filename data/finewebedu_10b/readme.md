
## fineweb-edu 10B sample dataset

after running `prepare.py` (preprocess), this folder will contain:

- `train.bin`
- `val.bin`

this script follows `data/openwebtext/prepare.py` closely, but loads:

- dataset: `HuggingFaceFW/fineweb-edu`
- config: `sample-10BT`

reference:

- [FineWeb / FineWeb-Edu datasets by HuggingFaceFW](https://huggingface.co/datasets/HuggingFaceFW/fineweb-edu)
